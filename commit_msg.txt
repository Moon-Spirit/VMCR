loader+vulkan: inline 化有返回值入口, 修复 1.3 features 字段

1. gl_entry.cpp FWD_RET0/1/2 宏在 NDK clang 下产生
   "type void" 错误 (宏的 token 拼接问题, 不可靠).
   修复: 全部 inline 写出, 11 个有返回值入口.

2. glTexSubImage2D 有 9 个参数, 之前用 FWD8 (8 个) 编译失败.
   修复: 改用 FWD9.

3. vk_renderer.cpp 使用 VkPhysicalDeviceVulkan13Features
   的 .timelineSemaphore 字段, 但 NDK 27.x / Vulkan SDK 1.4
   的该 struct 已经没有这个字段 (被移到 1.2 features).
   修复: 使用 VkPhysicalDeviceVulkan12Features 启用
   timelineSemaphore + bufferDeviceAddress, 1.3 只保留
   dynamicRendering + synchronization2.
