# SM8635 设备手册

## 设备信息

| 项 | 值 |
| :--- | :--- |
| 型号 | SM8635 (Snapdragon 8s Gen 3) |
| GPU | Adreno 735 |
| CPU | Kryo (Cortex-X4 + A720 + A520) |
| 制造 | TSMC 4nm |
| 发布时间 | 2024 |

## Vulkan 1.3 特性

参见 `configs/vulkan_features.json` 中 `sm8635` 节点.

## 验证方法

```bash
adb push $ANDROID_NDK_HOME/sources/third_party/vulkan/android-binary/vulkaninfo /data/local/tmp/
adb shell /data/local/tmp/vulkaninfo --output json > vulkaninfo.json
python3 scripts/parse_vkjson.py vulkaninfo.json
```

## 已知驱动版本

| 版本 | 备注 |
| :--- | :--- |
| 512.450.0+ | 推荐, 全部 1.3 特性通过 |
| 512.440.0 | 大部分 OK, `dynamicRenderingLocalRead` 可能 panic |
| < 512.420 | 不支持, 需要升级 |
